# 6. Strings and Methods

### What is it?
Strings are text. They have built-in methods (tools) to manipulate them (like turning text UPPERCASE).

### Real-World Analogy
A label maker that can automatically capitalize names.

### Why/When to use it
Formatting user input.

### Syntax & Code
```java
String name = "dileep".toUpperCase();
```

### Dileepkumar Bank Example
We use `.startsWith("RU")` to detect sanctioned countries in the Fraud Engine.
