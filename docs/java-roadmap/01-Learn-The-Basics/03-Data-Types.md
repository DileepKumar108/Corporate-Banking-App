# 3. Data Types

### What is it?
Java is strongly typed. You must specify if a variable is an integer (`int`), decimal (`double`), text (`String`), or true/false (`boolean`).

### Real-World Analogy
You cannot put soup into a paper envelope.

### Why/When to use it
To prevent bugs.

### Syntax & Code
```java
int age = 30;
```

### Dileepkumar Bank Example
Balances are stored as `double` to hold decimal cents.
