# 8. Arrays

### What is it?
A fixed-size list of items of the same type.

### Real-World Analogy
An egg carton. It holds exactly 12 eggs. You cannot put a 13th egg in.

### Why/When to use it
Storing a known number of items.

### Syntax & Code
```java
int[] scores = {100, 95, 80};
```

### Dileepkumar Bank Example
Storing the last 5 login timestamps for a user.
