# 2. Lifecycle of a Program

### What is it?
Java code goes through a lifecycle: Write (.java), Compile (.class), and Run (JVM).

### Real-World Analogy
Writing a recipe, translating it to French, and the Chef cooking it.

### Why/When to use it
This is how Java achieves Write Once, Run Anywhere.

### Syntax & Code
```bash
javac Bank.java
java Bank
```
