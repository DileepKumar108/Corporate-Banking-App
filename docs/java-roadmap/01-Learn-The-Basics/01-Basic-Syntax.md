# 1. Basic Syntax

### What is it?
Syntax is the grammar of a programming language. In Java, every line of code must end with a semicolon `;`, and all code must exist inside a Class.

### Real-World Analogy
Like ending a sentence with a period.

### Why/When to use it
You must use it always, or the code will not compile.

### Syntax & Code
```java
System.out.println("Hello Bank!");
```
