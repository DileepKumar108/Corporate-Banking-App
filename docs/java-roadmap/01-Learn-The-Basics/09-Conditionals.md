# 9. Conditionals

### What is it?
If/Else statements that make decisions in your code.

### Real-World Analogy
A fork in the road.

### Why/When to use it
Anytime the code needs to make a choice.

### Syntax & Code
```java
if (balance < 0) { System.out.println("Overdrawn"); }
```

### Dileepkumar Bank Example
`if (dti > 40) { rejectLoan(); }`
