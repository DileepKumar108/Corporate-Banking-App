# 10. Loops

### What is it?
For and While loops repeat code until a condition is met.

### Real-World Analogy
Running laps around a track until you hit 5 miles.

### Why/When to use it
Processing hundreds of items without copy-pasting code.

### Syntax & Code
```java
for (int i=0; i<5; i++) { /* repeat */ }
```

### Dileepkumar Bank Example
Looping through an array of 1,000 transactions to calculate the total spent.
