# Northstar Bank Frontend

A premium-looking banking frontend built with React, Vite, Redux Toolkit, and Redux Saga.

## Features
- Modern login experience
- Secure-looking banking dashboard
- Responsive layout
- Clear modular architecture
- Demo state for rapid UI development

## Getting started
```bash
npm install
npm run dev
```

## Available scripts
- npm run dev - start the development server
- npm run build - create a production build
- npm run lint - run ESLint checks
